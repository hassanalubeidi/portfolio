Killer Penguins—WebGL gyroscope controlled FPS built for Hack The Holidays 2015. Won 3rd place. Github repo.

Goofy but technically complex, Killer penguins was built in 24 hours by a team of three.

My main contribution was to create a site that uses SocketIO and the experimental device orientation API to create a barebones Wiimote and nunchuck out of two smartphones.
